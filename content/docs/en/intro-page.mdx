Customermates is an open-source CRM designed to be driven by your AI. Native MCP support means Claude, ChatGPT, Cursor, and anything else that speaks the Model Context Protocol can read and write your CRM without copy-paste.

## What Customermates is

A modern CRM with the usual parts (contacts, organizations, deals, services, tasks, custom fields, pipelines, dashboards) plus:

- **Native MCP endpoint** so your AI client can discover and call CRM operations without a plugin.
- **Open source** under AGPLv3 (Enterprise components under a separate commercial license). Self-host or use our cloud.
- **EU-hosted and GDPR-compliant** for the managed version.
- **Real webhooks** with signed payloads, retries, and a delivery log.
- **OpenAPI 3.1** for the full REST surface.

## Four ways Customermates talks to your stack

| I want to… | Use |
|---|---|
| Have Claude, ChatGPT, or Cursor update the CRM in conversation | [MCP](/docs/mcp) |
| Build a custom integration in code | [OpenAPI](/docs/openapi) |
| React to CRM changes in Slack, a sequencer, BI, your warehouse | [Webhooks](/docs/webhooks) |
| Compose workflows visually without code | [n8n](/docs/n8n) |

All four use the same API key. See [API keys](/docs/api-keys).

## Our take

Most CRMs were designed for a world where humans typed into forms. That world is ending. Customermates treats the AI as a first-class user: tools are named imperatively, errors are actionable, relationships are safe to modify incrementally, and the surface is small enough for a weak model to navigate reliably.

The CRM that updates itself in the AI you already use. Not a new tab, not another app.
